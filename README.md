# CF Industries Equity Research Dashboard

Interactive stock pitch dashboard for the Perplexity Stock Pitch Competition 2026.

**Thesis:** LONG CF Industries (NYSE: CF) at $130.53, target $185 (+42% upside). CF is a second-order beneficiary of the Strait of Hormuz closure — cheap US natural gas ($2.89/MMBtu) gives CF the widest cost advantage in a decade while competitors face $15+ global LNG prices.

**Live dashboard:** [cf-industries-research](https://www.perplexity.ai/computer/a/cf-industries-institutional-eq-C4QM6QzQSxeVEYiOkN9PMQ)

## Pages

1. **The Big Picture** — Investment thesis, criteria checklist, interactive investment simulator
2. **War & Oil** — Hormuz crisis timeline, prediction markets, shipping disruption data
3. **Why Wall Street Is Wrong** — Variant perception analysis, scenario modeling, kill conditions
4. **Company Deep Dive** — Quarterly financials, segment breakdown, analyst ratings, peer comparison
5. **The Trade** — Position structure, risk management, catalyst calendar, macro backdrop

## Data Sources

- Prediction markets: Polymarket, Kalshi
- Shipping intelligence: Lloyd's List, Kpler, Windward
- OSINT: ACLED, ISW Critical Threats
- Macro: BLS, FRED
- Financial data: Perplexity Finance tools

## Stack

React, Vite, Tailwind CSS, Recharts. Design adapted from [pgh-transit-atlas](https://rzrizaldy.github.io/pgh-transit-atlas/) — brutalist aesthetic with JetBrains Mono, thick borders, no rounded corners.

```

```
